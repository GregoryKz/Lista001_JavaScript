# Lista 01 de Exercícios – Operadores Aritméticos

## Descrição
Esta lista contém 19 exercícios de programação em **JavaScript**, focados em operadores aritméticos e manipulação de dados.  
Os exercícios vão desde operações básicas, como soma e subtração, até cálculos aplicados, como área de figuras geométricas, cálculo de IMC e conversão de unidades.

Cada exercício está organizado em uma pasta separada (`exercicio001`, `exercicio002`, etc.), contendo um arquivo `index.html` e a lógica em JavaScript dentro da pasta `js`.

## Lista de Exercícios
1. Solicitar o nome do usuário e exibir uma saudação.  
2. Somar dois números digitados pelo usuário.  
3. Subtrair dois números digitados pelo usuário.  
4. Multiplicar dois números digitados pelo usuário.  
5. Dividir dois números digitados pelo usuário.  
6. Exibir o sucessor de um número.  
7. Exibir o antecessor de um número.  
8. Calcular a área de um retângulo.  
9. Calcular a área de um triângulo.  
10. Calcular a área de um quadrado.  
11. Calcular a área de um círculo.  
12. Calcular a média aritmética de três notas.  
13. Exibir o dobro de um número inteiro.  
14. Exibir a parte inteira de um número real.  
15. Calcular o Índice de Massa Corporal (IMC).  
16. Aplicar 10% de desconto em um produto.  
17. Calcular a média das idades de três pessoas.  
18. Converter metros para centímetros e milímetros.  
19. Calcular a hipotenusa de um triângulo.  

---

## Como Usar
1. Abra a pasta do exercício desejado.  
2. Abra o arquivo `index.html` em seu navegador.  
3. Insira os dados solicitados e veja o resultado.
